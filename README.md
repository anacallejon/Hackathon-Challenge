About This Project

This interactive 404 page is based on a narrative concept where a hostile artificial intelligence has taken control of the navigation system. Instead of simply “breaking” a link, the AI has started erasing routes across the cosmos, removing any paths it considers unnecessary—or annoying.
The user has accidentally fallen into one of those erased routes.

Throughout the page, the AI (named OBSIDIAN) communicates through glitched visual frames and distorted speech bubbles, gradually revealing the situation with a sarcastic, unstable tone. As the user scrolls, OBSIDIAN’s expressions shift, the interface glitches, and the story progresses until the AI eventually forces a full reset and declares:

“Welcome to Mars.”

This project combines:
-A scroll-driven narrative
-Animated glitch effects
-A dynamic starfield canvas background
-Responsive layout and behavior changes for mobile
-Smooth transitions between the main content and the final message

The goal was to transform a standard 404 error page into a small cinematic experience, where the user interacts with a malfunctioning AI who has literally deleted the route they were trying to reach—and doesn’t seem to regret it.
